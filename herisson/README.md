# Lancement

## Lancement locale

### Installer pm2
`npm install pm2 -g`

### Installer les modules dans la racine et le dossier herisson
`npm install`

### Lancer le serveur dans la racine : 
`pm2 start server.js --name herisson`

Aller sur http://127.0.0.1:8080/herisson/
